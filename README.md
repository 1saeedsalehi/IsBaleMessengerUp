# Bale uptime

[![Telegram](https://img.shields.io/badge/Telegram-@isbaleup-26A5E4?logo=telegram&logoColor=white)](https://t.me/isbaleup)

Tired of refreshing to see whether [Bale](https://web.bale.ai/) is reachable? This repo probes the site every **30 minutes** (GitHub Actions) and posts to Telegram channel when it’s back.

**فارسی:** اگر از اینکه هی خودت چک کنی ببینی پیام رسان «بله» وصل است یا نه خسته شدی، این به دردت میخوره. هر نیم ساعت یک‌بار چک می‌کنیم ببینیم بله روی وب بالا می‌آید یا نه؛ فقط وقتی وضعیت عوض شود در  [کانال تلگرام](https://t.me/isbaleup) خبر می‌دهیم.
